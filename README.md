### 🚀 Experience & Key Projects

#### **ML Developer @ Sber (May 2025 – Present)**
*Focus: Internal AI products, NLP pipelines, and VLM systems.*

* **🛡️ Secure AI Tech Support Ecosystem**
    * Designed a system with advanced **PII anonymization** for incoming/outgoing data.
    * Implemented a **GraphRAG** knowledge base for deep context retrieval.
    * Integrated seamlessly with Service Desk APIs for automated ticket handling.

* **🏙️ Citizen Support Agent (Stateless Architecture)**
    * Built a high-load AI agent for housing & utility services (ZhKH) using **LangFlow** & **LangGraph**.
    * Engineered a strictly **stateless architecture** for horizontal scalability.
    * Developed smart address parsing using custom **NER** models and Dadata integration.

* **👁️ Automated VLM Labeling Pipeline**
    * Created a zero-shot labeling system for complex defect detection.
    * Orchestrated a pipeline using **DINOv2, SAM, and CLIP** to auto-annotate datasets.
    * Used the output to train lightweight **YOLO** models, eliminating manual labeling costs.

* **📊 Strategic Analytics & Education**
    * Developed local AI solutions for analyzing global strategic trends.
    * Conducted corporate workshops and lectures on LLMs, preparing theoretical materials and practical cases for employees.
